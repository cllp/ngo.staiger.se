# ngo.staiger.se

Humanitarian & Environmental Technology Portfolio — Claes-Philip Staiger

## About

Personal portfolio website positioning 20+ years of enterprise architecture expertise toward humanitarian response, health equity, and environmental resilience. Featuring experience with MSF International, UNESCO, and major enterprise clients.

## Stack

- Static HTML/CSS/JS (single file, zero dependencies)
- Google Fonts (Inter + Playfair Display)
- Dark mode support
- Mobile-first responsive design
- Accessibility-first

## Deployment

Hosted on GitHub Pages with custom domain.

### DNS Setup

Add CNAME record to staiger.se DNS:

```
CNAME  ngo  →  [username].github.io
```

### GitHub Pages

1. Go to repository Settings → Pages
2. Source: Deploy from branch `main`
3. Custom domain: `ngo.staiger.se`
4. Enforce HTTPS: ✓

## Development

No build step required. Open `index.html` in a browser to preview.
